# QR
Flask WebApp + HTML page for QR Code generation

Step 1 : Run the flask sever :  ```python server.py```.<br>
Step 2 : Open ```index.html``` in a browser window.<br>
Step 3 : Type the data you want to put and press "Generate QR".<br>
